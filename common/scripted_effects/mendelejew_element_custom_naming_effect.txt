# For Element 63, I chose the approach of making the OTL name the default, and striking out almost all european culture triggers above it, to avoid a list of 50 odd
# cultures for a single check, as that may have impacted performance. An exception was made however, for Gallium, Germanium, Ruthenium, Italicum, Hibernium, Polonium,  
# Coppernicum, Hellenicum, Helvetium, Belgicum, Hafnium, Holmium and Icenium. This is because those 13 elements, with the exception of Coppernicum, are the dominant 
# elements for that group, that is, except when there is a specific reason to switch up the ordering, they are the first in the list for their respective culture, 
# and thus their existance is a reasonable proxy for the question of whether a particular nation has gotten an element named after them or not. 

# Exceptions for the default list ordering are based on otl element names, so for example, Francium would be above Gallium for element 87, as that was the one that
# got named Francium in our timeline. List order can also be modified on the basis of 'mirrors' of otl names, so for example, Sequana (latin name for the Seine river) 
# is at the bottom of the French list for most elements, but is at the top for Element 75, which in otl is named Rhenium after the river rhine.
